Project

Project Title: E-Commerce Website

Project Description:


This e-commerce website project demonstrates my front-end web development skills in creating a modern and user-friendly online store. The website includes multiple pages, such as the homepage, product listing page, product detail page, and shopping cart.

Technologies Used:


HTML5
CSS3
GitHub Link:https://github.com/Yelimbeti/e-commerce.git

Pages:


Homepage:
A visually appealing homepage showcasing featured products and promotional banners. Navigation bar with links to different product categories and the shopping cart. Footer with contact information and links to social media.
Product Listing Page:
A grid-based layout displaying various products in the selected category. Each product card includes an image, product name, price, and "Add to Cart" button.
Product Detail Page:
Detailed information about a selected product, including larger product images, product description, price, and customer reviews. Option to select product variants (e.g., color, size) if applicable. "Add to Cart" button to add the product to the shopping cart.
Shopping Cart:
A page that displays the list of products added to the shopping cart. Ability to update the quantity or remove items from the cart. Show the total price of all items in the cart.

Key Features:


Responsive Design:
The website is optimized for various screen sizes, including desktop, tablet, and mobile devices.
CSS Styling:
Custom CSS styles have been applied to enhance the appearance and user experience.
Interactive Elements:
Buttons and links change appearance on hover to provide feedback to users.
